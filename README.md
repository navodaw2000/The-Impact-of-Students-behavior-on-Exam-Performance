# The Impact of Students’ Behavior on Exam Performance using R Programming

## STAT 31631 – Statistical Modeling
**Department of Statistics & Computer Science, University of Kelaniya**  
**Academic Year:** 2023/2024  

### Description:
This project investigates how students’ behavior impacts exam performance using R programming. We analyze factors like study habits, attendance, and participation using statistical modeling techniques such as regression analysis and correlation analysis.  

### Repository Structure:
- **Data/** – Contains the dataset used for analysis
- **RMarkdown/** – Analysis R Markdown file and knitted HTML output
- **Report/** – Final PDF report

### How to Run:
1. Open `analysis.Rmd` in RStudio.
2. Knit to PDF or HTML to reproduce the analysis.
